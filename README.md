# Generative Art

## Deformed Penrose Tiling

Exploring a method of shading an image by mapping the lightness of a pixel to the amount of the "bright" tile to show. Makes for some neat animations.

Interactive demo on my personal site [here](https://willrobertson.id.au/projects/deformed-penrose-tiling-16).

Inspiration: [u/blind_firefly on Reddit](https://www.reddit.com/r/generative/comments/eexh33/penrose_tile_deformations/)

Concept of deforming penrose tiles: [Deformed Penrose tiling and quasicrystals - Richard Welberry](https://www.iucr.org/news/newsletter/volume-27/number-3/deformed-penrose-tiling-and-quasicrystals)

Starting point for code (ported and extended Python to P5.js): [Penrose Tiling Explained - Jeff Preshing](https://preshing.com/20110831/penrose-tiling-explained/)
